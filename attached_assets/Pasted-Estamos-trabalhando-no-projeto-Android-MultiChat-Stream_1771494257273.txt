Estamos trabalhando no projeto Android MultiChat Stream Viewer, desenvolvido em:

Kotlin

Jetpack Compose

MVVM

Room

Hilt

Coroutines + Flow

O app já possui:

Cadastro de múltiplos chats

WebView por chat

Modo multichat (várias janelas simultâneas)

Sistema de autenticação condicional

Atualização em tempo real

🎯 Objetivo da Tarefa

Adicionar uma nova funcionalidade opcional chamada:

Modo Chat Unificado

Quando ativado, o app deve:

Combinar todas as mensagens dos chats ativos

Exibir tudo em UMA única janela

Manter identificação visual de origem (plataforma + nome do chat)

Ordenar mensagens por timestamp

Atualizar em tempo real

Sem remover ou quebrar o modo multichat atual.

1️⃣ Toggle de Ativação

Adicionar um toggle na tela Multichat:

Switch: “Modo Unificado”

Persistir escolha no Room ou DataStore

Alternância dinâmica sem reiniciar tela

2️⃣ Estrutura de Dados

Criar modelo:

data class UnifiedMessage(

    val message: String,
    val timestamp: Long
)

3️⃣ Agregação de Mensagens

Cada ChatWebViewComponent já emite mensagens via Flow.

Um método que combine múltiplos Flows usando:

combine()


Emitir lista unificada ordenada por timestamp

Limitar histórico a 500 mensagens
