Crie um aplicativo Android cujo objetivo é permitir que streamers visualizem simultaneamente o chat de múltiplas transmissões ao vivo em uma única tela, permitindo que o usuário adicione manualmente URLs de chats de diferentes plataformas de streaming.
O aplicativo deve ser leve, rápido, simples e totalmente funcional sem necessidade de login, cadastro ou servidor externo.
🎯 Objetivo do Aplicativo
Permitir que streamers que transmitem para várias plataformas consigam acompanhar todos os chats em uma única tela do celular, servindo como substituto de um segundo monitor.
O aplicativo deve funcionar localmente no dispositivo e não deve exigir criação de conta.
🧩 Funcionalidades Obrigatórias
➤ Cadastro de Chats
O usuário deve conseguir adicionar novos chats manualmente informando:

URL do chat ou live
Nome personalizado do chat
Plataforma (opcional)
Opção de ativar ou desativar o chat
O sistema deve permitir cadastro ilimitado de chats.
➤ Gerenciamento de Chats
O aplicativo deve possuir uma tela para gerenciamento contendo:

Lista de chats cadastrados
Opção de editar chat
Opção de remover chat
Opção de reordenar chats
Opção de ativar/desativar chats temporariamente
➤ Visualização Individual do Chat
Cada chat deve poder ser aberto separadamente contendo:

Exibição das mensagens em tempo real
Nome do usuário da mensagem
Mensagem enviada
Avatar do usuário (quando disponível)
Identificação visual da plataforma
Scroll automático com opção de pausar
➤ Visualização Multichat (Modo Principal do App)
O aplicativo deve permitir visualizar todos os chats simultaneamente em uma única tela.

O usuário deve poder escolher:
Layout em colunas
Layout em grade
Layout em lista vertical
Cada chat deve funcionar de forma independente dentro da tela.
➤ Atualização em Tempo Real
Os chats devem atualizar automaticamente utilizando:

WebView incorporado

OU
APIs públicas quando disponíveis
Não deve exigir atualização manual da tela.
➤ Identificação Visual
Cada chat deve exibir:

Nome do chat
Ícone da plataforma
Destaque visual para diferenciar plataformas
➤ Modo Streamer (Prioridade de Uso)
O aplicativo deve ter um modo otimizado para transmissão contendo:

Interface limpa
Botões ocultáveis
Tela sempre ativa (evitar bloqueio do celular)
Ajuste rápido de zoom e tamanho da fonte
Opção de fixar chats importantes
💾 Armazenamento Local
O aplicativo deve salvar todos os dados localmente no dispositivo utilizando:

SQLite ou Room Database
Os dados que devem ser salvos incluem:

Chats cadastrados
Configurações do usuário
Layout selecionado
Ordem dos chats
Os dados devem persistir mesmo após fechar o aplicativo.
